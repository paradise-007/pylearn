## ✨ Features

| Feature | Description |
|---|---|
| 📦 **Course Modules** | 12 structured modules from Python basics to testing |
| ⌨️ **Code Editor** | In-browser editor with simulated execution and test cases |
| 🧠 **Quiz Arena** | 10-question quiz with instant feedback and XP rewards |
| 🚀 **Project Lab** | 9 guided projects sorted by difficulty level |
| 🎬 **Video Tutorials** | Curated YouTube links for every topic |
| 📚 **Docs & Resources** | Links to official docs, cheatsheets, and practice sites |
| 💬 **Community Forum** | Ask questions and share solutions with learners |
| 📈 **Progress Tracking** | XP system, completion rings, weekly activity chart |
| 🏅 **Badges** | 15 achievements to unlock as you learn |
| 🌙 **Dark / Light Mode** | Full theme toggle with smooth transitions |
| 🔍 **Search** | Instant search across all modules and topics |
| 📱 **Mobile Friendly** | Fully responsive with collapsible sidebar |

## 🚀 Getting Started

### Option A — Use it instantly (no setup)
👉 **[Open Live App](https://yourusername.github.io/pycompanion)**

### Option B — Run locally
```bash
# Clone the repository
git clone https://github.com/yourusername/pycompanion.git

# Navigate into the folder
cd pycompanion

# Open directly in your browser
open index.html

# OR serve with Python's built-in server
python3 -m http.server 8000
# Then visit http://localhost:8000
```

## 📚 Curriculum Overview

### 🟢 Beginner
- Python Basics — variables, types, operators
- Control Flow — if/elif/else
- Loops — for, while, range()

### 🟡 Intermediate
- Functions — def, scope, *args, **kwargs
- Data Structures — lists, dicts, sets, tuples
- List Comprehensions
- File I/O
- Error Handling

### 🔴 Advanced
- Object-Oriented Programming
- Modules & Packages
- APIs & Requests
- Testing with pytest
