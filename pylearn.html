⑂ Git & Version Control

Master Git — the industry standard version control system. Track changes, collaborate, and never lose your work.

bash — ~/myproject
$ git init
Initialized empty Git repository in ~/myproject/.git/

$ git add .
$ git commit -m "Initial commit"
[main (root-commit) a1b2c3d] Initial commit
3 files changed, 45 insertions(+)

$ git status
On branch main
nothing to commit, working tree clean
Essential Commands

git init

Initialize a new Git repository in your project directory. This creates the hidden .git folder that tracks all changes.

git add & commit

Stage changes with git add, then save a snapshot with git commit -m "message".

git branch

Create isolated branches for features. Merge them back to main when ready with git merge.

git push / pull

Sync your local repository with GitHub or other remotes using push and pull commands.

💡 Pro Tip: Write meaningful commit messages in the imperative mood: "Add login feature" not "Added login feature". Your future self will thank you!
